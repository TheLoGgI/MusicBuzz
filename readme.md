# Music Buzz

## Development

**Run React App**

```
cd app && pnpm dev
```

**Run Go API**

```
go run main.go
```

## Production

Build react app

```
cd app && pnpm build
```

**Run Go API**

```
go run main.go
```

**Hosting on Vercel**

All files in the api/ folder is routes for the Vercel, no need to do anything.
