# Music Buzz

## Development

Export envs' in .env

**Run React App**

```
cd app && pnpm dev
```

**Run Go API**

```
go run main.go
```

## API's

Testing api with Postman for websocket you need to set the Origin Header to the Domain of the api.

- Connect to Websocket

## Production

Build react app

```
cd app && pnpm build
```

**Run Go API**

```
go run main.go
```

**Hosting on Vercel**

All files in the api/ folder is routes for the Vercel, no need to do anything.
